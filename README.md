Install script to set up netbird

How to use:

Running the install script

```sudo ./install.sh```

Running the uninstall script

```sudo ./uninstall.sh```

UPDATE INFO:

Updating the netbird version:

1. Change the NETBIRD_VERSION variable in install.sh to the desired version (e.g. 1.9.7)

2. run uninstall.sh

3. run install.sh
